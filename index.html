About Me

Hello World! I am an undergraduate computer science student at the University of Washington. I am interested in Machine Learning and Computer Vision and passionate about solving real problems with them. My work explores the intersection of deep neural networks and health.


Feel free to look through my projects below or by clicking the 'Projects' button on the top right.

Projects

Feature Extraction Architecture Evaluation for Metastatic Tissue

Focuses on the binary classification of histopathology scans to detect metastatic vs. non-metastatic tissue evaluating multiple state-of-the-art CNN architectures as feature extractors.

Vaccine Reservation System

A Java application for managing vaccine scheduling. It securely connects to a local SQLite or a remote Amazon Aurora database to keep track of patients, caregivers, and vaccine appointments.

Campus Shortest Path Finder

A full-stack web application built with React and Node.js that allows users to find the shortest walking path between buildings on campus using Dijkstra's algorithm and an interactive map interface.

Interactive 3D Audio Visualization Tool

A real-time 3D visualization built in JavaScript that responds dynamically to live microphone input using the p5.js library, extracting frequency bands to map onto geometric elements.
